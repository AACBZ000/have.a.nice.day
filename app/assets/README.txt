Destiny Pillars - App Assets

REQUIRED before building for App Store:
----------------------------------------

1. icon.png
   - Size: 1024x1024 pixels
   - Format: PNG, no transparency
   - Content: Destiny Pillars app icon
   - Design suggestion: Taijitu (yin-yang) symbol on deep navy (#0D1B2A) background with gold (#C9A84C) accents
   - Place as: app/assets/icon.png

2. splash.png (optional but recommended)
   - Size: 1284x2778 pixels (iPhone 14 Pro Max)
   - Or use 2048x2048 for universal
   - Content: App name / logo on navy background
   - Place as: app/assets/splash.png

3. favicon.png (for Expo web preview)
   - Size: 32x32 pixels
   - Place as: app/assets/favicon.png

4. adaptive-icon.png (Android)
   - Size: 1024x1024 pixels, with safe zone of 66% centered
   - Place as: app/assets/adaptive-icon.png

Tools to create icons quickly:
- https://www.canva.com
- https://appicon.co (upload your 1024x1024 and it generates all sizes)
- https://makeappicon.com

For a temporary placeholder during development, you can use any 1024x1024 PNG
renamed to icon.png — it only matters for App Store submission.
